## Português

### Novidades

- Desistência nos desafios: depois de um desafio começar já não se sai da sala — desiste-se, o que o dá como perdido. Os pontos já feitos ficam registados, mas quem desiste cai para último lugar e fica marcado no placar e no histórico.
- Proteções contra abuso da desistência: só depois de jogar pelo menos uma partida (ou 15 minutos de espera), e com uma hora de intervalo entre desistências.
- Botão "Terminar desafio" para o anfitrião — fecha com as pontuações atuais um desafio em que alguém deixou de aparecer, em vez de ficar preso para sempre.
- Regras específicas a funcionar na criação da sala: handicap por arquétipo, só KDA, e só jogos sem adversários do desafio na mesma Arena.
- Botão "Voltar aos desafios" quando o desafio termina — volta ao menu mantendo o desafio guardado no histórico.

### Melhorias

- Placar do desafio reorganizado em função do que pontua: cada estatística mostra o valor e, entre parênteses, os pontos que rendeu; categorias que não contam nas regras da sala deixam de aparecer.
- Pontuação de sequências escalonada: cada abate acima de 3 seguidos vale mais, a subir de nível de 2 em 2 (5 abates seguidos = 2+2+2+3+3 = 12 pontos). O mesmo para assistências, e para mortes seguidas — a penalizar.
- Handicap por arquétipo: 8 tipos de campeão (tanque, engage, juggernaut, duelista, assassino, atirador, mago, encantador) com valores afinados a partir das partidas reais registadas na app, para todos os estilos renderem em média o mesmo.
- O histórico de desafios passa a indicar com que regras cada um foi pontuado.

### Correções

- Um desafio já não pode ser vencido sem se ter jogado nenhuma partida — se todos os outros desistirem antes de o líder jogar, o desafio termina sem vencedor.

## English

### What's New

- Challenge forfeits: once a challenge starts you no longer leave the room — you forfeit, which counts as a loss. Points already scored are kept, but the forfeiter drops to last place and is flagged on the scoreboard and in history.
- Anti-abuse protections on forfeiting: only after playing at least one match (or a 15-minute wait), with a one-hour gap between forfeits.
- "End challenge" button for the host — closes a challenge someone abandoned with the current scores, instead of leaving it stuck forever.
- Custom rules now working when creating a room: archetype handicap, KDA only, and only games without challenge rivals in the same Arena.
- "Back to challenges" button once a challenge ends — returns to the menu while keeping the challenge saved in history.

### Improvements

- Challenge scoreboard reorganised around what actually scores: each stat shows its value and, in brackets, the points it earned; categories that don't count under the room's rules are hidden.
- Escalating streak scoring: each kill past 3 in a row is worth more, levelling up every 2 (5 kills in a row = 2+2+2+3+3 = 12 points). Same for assists, and for death streaks — as a penalty.
- Archetype handicap: 8 champion types (tank, engage, juggernaut, skirmisher, assassin, marksman, caster, enchanter) with values tuned against the real matches recorded in the app, so every playstyle scores the same on average.
- Challenge history now shows which rules each challenge was scored with.

### Fixes

- A challenge can no longer be won without playing a single match — if everyone else forfeits before the leader plays, the challenge ends with no winner.
